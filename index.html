贪吃蛇玩法

1. 游戏目标：控制蛇吃到食物，让蛇变得更长并获得高分。

2. 控制方式：方向键或WASD控制蛇的移动方向。

3. 游戏规则：撞到墙壁或自己的身体会导致游戏结束。

4. 得分规则：每吃一个食物得10分，蛇的长度会增加。

5. 难度递增：随着分数增加，蛇的移动速度会逐渐加快。

俄罗斯方块玩法

1. 游戏目标：将下落的方块填满一行，消除该行并获得分数。

2. 控制方式：左右方向键移动方块，上键旋转方块，下键加速下落。

3. 游戏规则：方块堆到顶部时游戏结束。

4. 得分规则：同时消除的行数越多，得分越高。

5. 技巧：尽量消除更多行数，保持方块堆高度较低。

连连看玩法

1. 游戏目标：找出所有成对的相同图案并消除它们。

2. 操作方式：点击两个相同的图案，如果它们之间有可连接的路径则消除。

3. 连接规则：两个图案之间的连线不能超过三条直线。

4. 胜利条件：在规定时间内消除所有图案。

5. 提示功能：遇到困难时可以使用提示功能，但会扣除一定分数。

扫雷游戏玩法

1. 游戏目标：找出所有非地雷的格子，避免踩雷。

2. 左键点击：揭开格子，如果是地雷则游戏结束。

3. 右键点击：标记疑似地雷的格子。

4. 数字提示：格子上的数字表示周围8个格子中地雷的数量。

5. 胜利条件：成功揭开所有非地雷格子。
